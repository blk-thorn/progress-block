# Блок Progress

## Задача
Разработать прототип блока Progress для использования в мобильных web-приложениях.
Основное предназначение блока отображать процесс выполнения процессов и их прогресс выполнения.

### Ссылка на деплой: https://blk-thorn.github.io/progress-block/src

Основной функционал:
Normal – базовое состояние, в котором путём задания Value можно управлять размером дуги. Начало дуги соответствует 12 часам, конец дуги при увеличении параметра Value движется по направлению часовой стрелки и достигает начала при значении 100.
Animated – независимое состояние, при котором блок или его элементы начинают вращаться по часовой стрелке.
Hidden – состояние скрывающее блок со страницы.

Приложение адаптируется под ориентацию экрана(portrait, landscape)

Дополнительный функционал:
Добавлено отображения процентов внутри прогресс-бара.
Добавлен фокус на инпут.
После остановки анимации дуга возвращается в исходное положение, соответствующее 12 часам.

Описание:
Код находится внутри папки src и разбит на модули progress-bar и progress-block, которые лежат в папке modules.
